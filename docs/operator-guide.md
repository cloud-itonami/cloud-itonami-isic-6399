# Operator Guide

## First Deployment
1. Register operator, source feeds, jurisdictions and personnel.
2. Import historical posting records.
3. Run read-only validation and publication dry-runs.
4. Configure jurisdiction catalogs, consent registers and human sign-off paths.
5. Publish a dry-run audit export.

## Minimum Production Controls
- governor gate on every publication and delisting before it reaches the public index
- human sign-off for every publication and delisting (handling employer/source data and public-facing advertisements)
- audit export for every publication, delisting, hold and sign-off
- backup manual process
- jurisdiction registration where required (e.g. Japan's 特定募集情報等提供事業者届出) is the operator's own obligation

## Certification
Certified operators must prove posting-accuracy integrity (的確表示),
evidence-backed records and human review for public-facing actions.

## Operating states
ingest : assess : publish : refresh : delist : audit
