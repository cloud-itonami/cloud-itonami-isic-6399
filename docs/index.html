Meta Job Search governed

The problem: Job boards lock in operator data or inject ads that exploit both job seekers and employers. The solution: An open-source job-posting aggregator you fork, operate, and control—with an independent Governor that automatically enforces compliance so illegal, discriminatory, or stale postings never reach searchers. Code · Business Model.

What this is NOT: This software does not crawl sources, match candidates, or place applicants—it aggregates postings, assesses them against jurisdiction rules, and publishes only those that pass an independent regulatory Governor.

Handoff to placement — human-carried referrals (ADR-2607131000)

Applications to board postings are not direct actor-to-actor calls, but human-carried referral records into the Placement Desk (isic-7810). Referrals require the applicant's own consent and hold only an applicant reference ID—no PII enters this public-facing store. The placement system is separate; both ledgers join at the record ID.

What this index guarantees

  • No stale postings: If the source employer filled the vacancy, it cannot be published (required by Japan's 的確表示義務 /職業安定法5条の4, USA's wage-law, UK/EU equivalent)
  • No pay misstatement: Displayed compensation is independently recomputed from source fields and always matches
  • No unlicensed republication: Postings from sources that require republication consent are held until verified
  • No discriminatory ads: Postings with protected-characteristic bias (age, gender, disability, race) are blocked (EEOC Title VII / UK Equality Act 2010 / Japan Equality Law / AGG §11)
  • Full audit trail: Every publication, delisting, and rejection is recorded in an immutable ledger—regulators can verify it, operators can prove compliance
