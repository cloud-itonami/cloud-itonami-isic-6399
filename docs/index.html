この検索インデックスが保証すること

  • 求人元が募集終了した求人は載らない(的確表示義務 — 職業安定法5条の4、令和4年改正)
  • 表示賃金は求人元記録からの独立再計算と常に一致する
  • 転載許諾が必要なソースの求人は、許諾確認なしに載らない
  • 保護属性に基づく差別的広告は載らない(均等法5条 / Title VII §704(b) / Equality Act 2010 / AGG §11)
  • すべての掲載・取下げ・拒否が追記専用の監査台帳に残る
